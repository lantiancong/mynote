

### apt软件包管理器安装

```bash
#查看是否存在apt
apt --version

#安装
sudo apt install 软件名称
#卸载
sudo apt purge 软件名称
sudo apt autoremove
```

查看软件是否存在

```bash
--version	或	-v
which 软件名   # 例如：which firefox
dpkg -l 软件名
type 命令名 
```

```bash
--version	或	-v
python3 --version  # 查看Python版本
git --version      # 查看Git版本
gcc -v             # 查看GCC版本（部分软件用小写-v）
-----------------------------------------------------------
which 软件名   # 例如：which firefox
-----------------------------------------------------------
#Debian/Ubuntu中
dpkg -l 软件名   # 例如：dpkg -l firefox
-----------------------------------------------------------
#判断命令是否存在（包括别名、内置命令等）：
type 命令名   # 例如：type ls
```



- 如果不执行 `update`，`apt` 可能会使用旧的索引信息，导致你安装的软件不是最新版本，或者找不到某些新发布的软件包

- ```bash
  sudo apt update
  sudo apt upgrade -y
  ```

### 寻找命令

```bash
基本语法：
bash
find 查找路径 查找条件 动作
常用示例：
按文件名查找（精确匹配）：
bash
find /home -name "test.txt"  # 在/home目录下查找名为test.txt的文件（区分大小写）
find /home -iname "test.txt"  # 不区分大小写（Test.txt、TEST.TXT也会被找到）
按文件名模糊查找（通配符 * 表示任意字符）：
bash
find /etc -name "*.conf"  # 在/etc目录下查找所有以.conf结尾的文件
find ~ -name "java*"  # 在当前用户目录下查找所有以java开头的文件/文件夹
按文件类型查找：
bash
find / -type f -name "*.sh"  # 查找所有后缀为.sh的普通文件（f表示文件，d表示目录）
find /var -type d -name "log*"  # 查找/var下所有以log开头的目录
按大小查找：
bash
find / -size +100M  # 查找大于100MB的文件（+表示大于，-表示小于，不加符号表示等于）
find ~ -size 10k  # 查找大小为10KB的文件（单位：b字节，kKB，MMB，GG B）
按修改时间查找：
bash
find / -mtime -7  # 查找最近7天内修改过的文件（-7表示7天内，+7表示7天前）
find /etc -mmin +30  # 查找30分钟前修改过的文件（mmin以分钟为单位）
```

```bash
grep "搜索内容" 文件名
常用示例：
在单个文件中查找：
bash
grep "error" /var/log/syslog  # 在syslog日志中查找包含error的行
在多个文件中查找：
bash
grep "java" /home/*.txt  # 在/home下所有.txt文件中查找包含java的行
递归查找目录下所有文件（-r 表示递归）：
bash
grep -r "password" /etc/  # 在/etc目录下所有文件中递归查找包含password的行
忽略大小写（-i）和显示行号（-n）：
bash
grep -in "root" /etc/passwd  # 在passwd中查找包含root的行，忽略大小写并显示行号
```

## 文本编辑器

### nano

- **粘贴内容**：从其他地方复制后，在 nano 中按 `Shift + Insert` 粘贴（部分系统支持右键粘贴）

| 操作                | 快捷键                | 说明                                                    |
| ------------------- | --------------------- | ------------------------------------------------------- |
| **保存文件**        | `Ctrl + O`            | 按后会显示文件名，回车确认保存                          |
| **退出编辑器**      | `Ctrl + X`            | 若文件未保存，会提示是否保存（输入 `Y` 确认，`N` 放弃） |
| **撤销操作**        | `Ctrl + _`            | 下划线需按 `Shift + -`，可多次撤销                      |
| **搜索文本**        | `Ctrl + W`            | 输入关键词后回车搜索，按 `Alt + W` 找下一个             |
| **替换文本**        | `Ctrl + \`            | 输入旧内容和新内容，批量替换                            |
| **跳到行首 / 行尾** | `Ctrl + A`/`Ctrl + E` | 快速定位到当前行的开头或结尾                            |
| **跳到指定行**      | `Ctrl + _`            | 输入行号后回车，直接跳转到对应行                        |
| **剪切当前行**      | `Ctrl + K`            | 剪切后，用 `Ctrl + U` 粘贴                              |
| **显示行号**        | `Alt + C`             | 临时显示光标所在行号                                    |
