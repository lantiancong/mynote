# 操作符

**>** 会覆盖

**>>** 不会覆盖,加在后面

**<**

**|**   前一个命令的输出作为后一个命令的执行对象

> ls -l | grep '.txt' 

**&** 后台运行

**&&**  		com1&&com2  ==>>com1为1, 才会执行com2

**||**   		com1&&com2  ==>>com1为0, 才会执行com2

**;**
