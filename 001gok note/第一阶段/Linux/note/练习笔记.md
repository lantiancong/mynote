# Linux练习

1. 打印“hello world”

   > echo

2. 查看当前工作目录路径

   > pwd

3. 列出当前工作目录内容

   > ls

4. 查看文件内容

   > cat

5. 输出某个文件倒数5行

   > tail -n file

6. 创建一个文件

   > touch

7. 创建一个文件夹（递归）：

   > mkdir -p(递归) tmp 【如果tmp存在会报错】
   >
   > mkdir -p tmp/files 【temp存在也不会报错】

8. 复制a.file到tmp文件夹

   > cp a.file tmp

9. 移动a.file到tmp文件夹

   > mv a.file tmp

10. 为a.file创建一个符号链接（软链接）

    > ln -s 【文件】 【链接名】（软）
    >
    > ln 【文件】 【链接名】（硬）
    >
    > 软==快捷方式；硬==同文件，不同名（inode相同）
    >
    > 源文件移动或删除，软失效（断链）

11. 删除该目录下的所有文件

    > rm -rf -- .[!.]* ..?* *
    >
    > “--” 防止误认后接命令参数
    >
    > "r"递归
    >
    > “f”强制
    >
    > [!.]非“.”
    >
    > “？”任意
    >
    > "*"任意个/通配
