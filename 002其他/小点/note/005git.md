### 克隆+拉取(pull)

克隆(clone)是第一次关联仓库时的执行的(**只有一次**)

后续想要更新仓库用的是 **git pull 别名 分支** 命令

```bash
# 强制拉取并覆盖本地更改
git fetch --all
git reset --hard origin/master
```
